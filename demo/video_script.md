# GuardianLens — Demo Video Script

> 70% of the score is impact + storytelling. Treat this file as the most
> important deliverable in the repo.

**Length:** strict 2:00 cap. Cut anything that does not move the story.
**Tools:** OBS Studio for capture, DaVinci Resolve / Shotcut for edit.
**Music:** quiet, tense, building. Drop on the first ALERT.

## Production checklist

- [ ] Stat overlay font: Inter / IBM Plex. White on black.
- [ ] Voice-over recorded twice; pick the calmer take.
- [ ] All chats in the demo are scripted — no real children.
- [ ] Three demo recordings:
  - [ ] safe_minecraft_chat.mp4
  - [ ] grooming_minecraft_chat.mp4
  - [ ] bullying_discord_chat.mp4
- [ ] Dashboard recorded at 1920x1080, 60 fps.
- [ ] Phone notification staged with the iPhone Notifications widget mockup.
- [ ] Final export: H.264 1080p, < 100 MB so it uploads fast.
- [ ] Backup upload: YouTube unlisted + Google Drive link.

## Things that ruin a 2-minute demo

- Reading the dashboard out loud. Let the viewer read.
- Talking head intro. The hook is the stat, not you.
- Fancy transitions. Hard cuts on the beat.
- Showing real child data. Always staged.
- Forgetting the GitHub link in the closing card.
